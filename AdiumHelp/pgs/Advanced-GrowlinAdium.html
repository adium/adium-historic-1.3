Growl in Adium

Adium version 0.80 and later have Growl support built-in, and since version 1.0, the default configuration has had Growl turned on for several events by default:

  • Contact signs on
  • Message received (Background Chat)
  • File transfer requested
  • File transfer completed successfully
  • File transfer failed
  • Error occurs

To turn some of these off, or to make Adium post additional Growl notifications, you need to set up one or more events (either on a contact, or globally in the Preferences) that post a Growl notification.

If Growl isn't installed, then the first time Adium tries to post a Growl notification, it will ask to install Growl. Adium 0.89.1 will install Growl 0.7.4.

If it still isn't working after you do that, open the Growl pane, and make sure Adium is listed on the Applications tab and that notifications from it are enabled.

If this setup is still not working for you, please visit our support forums at http://forum.adium.im.
