LaTeX in Adium

Note: Please see tickets #7592 and #10442 for information on the current status of this plugin.

Follow the steps under "Installing TeX" and under "Installing LaTeX Plugin".

Installing TeX

  • Download i-Installer.
  • Launch i-Installer.
  • Open the Known Packages i-Directory (shift-âŒ˜-O).
  • Select "gwTeX based on TeX Live" in i-Packages column.
  • Press "Open i-Package"
  • Press "Install & Configure"
  • Press "Basic".
  • You may be prompted to specify a default paper size.

i-Installer will install TeX and configure it automatically. Enter your administrator password whenever the installer requests it. You will be notified when the process has completed; you may quit the application and trash i-Installer at this point.


Installing LaTeX Plugin

Install the LaTeX Plugin from adiumxtras.


Using LaTeX Plugin

The LaTeX plugin will automatically parse equations between $$ markers, e.g. $$m = \frac{m_0}{\sqrt{1-\frac{v^2}{c^2}}}$$. (Alternatively \[ ... \] can be used as equation markers.) The rendered equation will be substituted into your message.
