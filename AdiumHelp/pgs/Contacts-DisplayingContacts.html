Displaying contacts

Adium allows you to determine how contacts are displayed and sorted in the contact list.

Sorting contacts

  • Sort Contacts Alphabetically
  • Sort Contacts by Status
  • Sort Contacts Manually

Further options

  • When sorting alphabetically, the Configure Alphabetical Sort... option in the View menu allows sorting by last name and sorting groups.
  • When sorting by Status, the Configure Status Sort... option in the View menu allows much customization, including sorting by status then manually or status then alphabetically, determining the order of statuses by dragging to reorder a list, and sorting mobile contacts separately

Showing/hiding offline contacts

By default, Adium hides contacts who are offline.

If you want to see a complete list of your contacts, including contacts who are not online at the moment, select Show Offline Contacts from the View menu or hit ⇧⌘H.

To change back to the default, do the same thing—the menu item is a toggle, so choosing it repeatedly changes the setting back and forth.

Note: If you have offline contacts hidden, Adium also hides any empty groups. Thus, if you want to drag & drop some contacts into an empty group, or delete the group, you must Show Offline Contacts first.
